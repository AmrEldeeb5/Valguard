# Icon and Branding Guidelines

## App Icon Design Preferences

### Color Palette
- Primary: Indigo (#6366F1, #4F46E5)
- Accent: Purple (#8B5CF6)
- Success: Green (#10B981)
- Background: Dark slate (#0F172A, #1E293B)

### Icon Style
- Modern, minimalist design
- Circular vault/crypto theme
- Include security indicators (lock, shield)
- Use corner brackets for tech aesthetic
- Maintain consistency with app's dark theme

### Platform-Specific Requirements
- **Android**: Use adaptive icons (API 26+) with separate background and foreground layers
- **iOS**: Provide 1024x1024 PNG assets
- Ensure icons work well on both light and dark backgrounds

## Branding
- App Name: CryptoVault (or CryptoWallet)
- Theme: Premium, secure, modern
- Visual Language: Tech-forward, trustworthy, sophisticated
